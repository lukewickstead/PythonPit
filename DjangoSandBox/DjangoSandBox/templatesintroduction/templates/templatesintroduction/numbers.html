Numbers & Templates

Sizing with widthratio

The width is: {% widthratio 50 100 100 %} {% widthratio 50 100 100 as width %} The width is: {{ width }}

Is non-remainder division with divisibleby

Is 21 dividable by 3? = {{ 21|divisibleby:"3" }} Is 21 dividable by 5? = {{ 21|divisibleby:"5" }}

File Size formats with filesizeformat

{{ 1|filesizeformat }} {{ 1024|filesizeformat }} {{ 1048576|filesizeformat }} {{ 1073741824|filesizeformat }}

Formatting floats with floatforma

11.1111 to 3dp = {{ 11.1111|floatformat:"3" }} 11.1111 to 0dp = {{ 11.1111|floatformat:"0" }} 11.1111 to -3dp = {{ 11.1111|floatformat:"-3" }} 11 to 3dp = {{ 11|floatformat:"3" }} 11 to -3dp = {{ 11|floatformat:"-3" }}

Numbers as an array with get_digit

123456789[2]= {{ 123456789|get_digit:"2" }}

Displaying telephone numbers with phone2numeric

+44 1752 HELLO becomes{{ "+44 1752 HELLO"|phone2numeric }}

Addition with the add tag

  • 2 + 1 = {{ 2|add:1 }}
  • '2' + '1' = {{ '2'|add:'1' }}
  • 'a' + 'b' = {{ 'a'|add:'b' }}
  • {{ onetwo }} + {{ onetwo }} = {{ onetwo|add:onetwo }}
