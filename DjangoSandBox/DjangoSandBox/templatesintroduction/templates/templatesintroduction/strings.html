Strings & Templates

Outputting Variables

Variables are outputted with {{ who }}

The uppercase, lowercase, capitalised and title tags

lower upper capfirst title
{{ who | lower }} {{ who | upper }} {{ who | capfirst }} {{ who | title }}

Escaping with addslashes

{{ "Simon's slashes"|addslashes }}

Space padding with ljust, center & rjust

"{{ "Hello"|ljust:20 }}" "{{ "Hello"|center:20 }}" "{{ "Hello"|rjust:20 }}"

Replacing with cut

"{{ "Hello"|cut:"el" }}"

Providing a default value

  • {{ NotProvided|default:"Nothing was provided here!!!" }}
  • {{ False|default:"The value was false!" }}
  • {{ None|default_if_none:"Only displayed if this is None" }}

The pluralize tags

  • The plural of colour is colour{{ colours|pluralize }}.
  • The plural of colour is colour{{ 2 | pluralize }}.
  • The plural of person is pe{{ 2|pluralize:"rson,ople"}}.

Truncating long messages

truncatechars truncatechars_html truncatewords truncatewords_html
{{ longmessage|truncatechars:20 }} {{ longboldmessage|truncatechars_html:20 }} {{ longmessage|truncatewords:3 }} {#Automatically closes any html tags which would have been removed#} {{ longboldmessage|truncatewords_html:3 }}

The spaceless tag compresses html

{% spaceless %} This is some text {% endspaceless %}

The templatetag allows outputting reserved symbols

  • {% templatetag openblock %} {% templatetag closeblock %}
  • {% templatetag openvariable %} {% templatetag closevariable %}
  • {% templatetag openbrace%} {% templatetag closebrace %}
  • {% templatetag opencomment %} {% templatetag closecomment %}

The verbatim is the raw or @ of template strings

{% verbatim %} {{this will output}}{% as it nothing has happened %} {% endverbatim %}

Breaking lines with linebreaks and linebreaksbr

{{ messagewithnewlines|linebreaks }} {{ messagewithnewlines|linebreaksbr }}

Adding word wrap

{#This does not seem to work for me#} {{ longmessage|wordwrap:5 }}

Adding line numbers

{{ messagewithnewlines|linenumbers|linebreaks }}

Number Count

There are {{ longmessage|wordcount }} words in "{{ longmessage }}"

Python string formatting with the stringformat tag

{#https://docs.python.org/2/library/stdtypes.html#string-formatting-operations#}
  • {{ 1234.567|stringformat:"E" }}
  • {{ 1234.567|stringformat:"F" }}
  • {{ 1234.567|stringformat:"d" }}

Outputting booleans

  • {{ False|yesno:"yeah,no,maybe" }}
  • {{ True|yesno:"yeah,no,maybe" }}
  • {{ None|yesno:"yeah,no,maybe" }}
  • {{ None|yesno:"yeah,no" }}
