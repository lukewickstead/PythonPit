Conditions & Templates

The if, elif, else & endif tags

{% if age < 50 %} Less than 50 {% elif age == 50 %} Equal to 50 {% else %} The else clause {% endif %}

The and & or tags

{% if age < 50 and age > 10 %} Less than 50 but more than 10 {% elif age == 50 or age == 51 %} Equal to 50 {% else %} The else clause {% endif %}

The not tag

{% if not false %} Not False {% endif %}

The in & not in tags

{% if 1 in numbers%} 1 is in {{ numbers }} {% endif %} {% if 11 not in numbers%} 11 is not in {{ numbers }} {% endif %}

The ifequal, else endifequal tags

{% ifequal valueone valueone%} {{ valueone }} and {{ valueone }} are equal {% else %} {{ valueone }} and {{ valueone }} are not equal {% endifequal %}

The ifnotequal, else endifnotequal tags

{% ifnotequal valueone valuetwo%} {{ valueone }} and {{ valuetwo }} are not equal {% else %} {{ valueone }} and {{ valuetwo }} are equal {% endifnotequal %}

The firstof tag outputs the first true (non zero) element

{#Can take booleans and ints#} {% firstof 0 0 0 1 "All were false" %} {% firstof 0 0 0 0 "All were false" %}

Filters allow piping results through multiple tags

{% filter lower|capfirst %}This text will be HTML-escaped.{% endfilter %} {#Filters can be used with other tags#} {% if numbers|length >= 5 %} There are more than five numbers! {% endif %}

Variables with the with tag

{% with count=10 %} There are {{ count }} item{{ 2|pluralize }} {% endwith %}

Parenthesis is not supported within templates so you must rely on operator precedence

The precedence is the same as python: or, and, not, in, ==, !=, <, >, <=, >=
